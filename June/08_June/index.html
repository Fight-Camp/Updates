Progress page — chart & native nav

Native iOS large title that collapses on scroll. Chart now shows daily weight dots, 3-day trending line, weight plan with dashed forecast, and a today marker.

  • Native large title collapses to inline on scroll
  • Settings avatar in nav bar
  • Chart legend below (Daily Weight, Trending Weight, Weight Plan)
  • Solid lines up to today, dashed forecast past today
  • Start / Current / Change stat row

Weight Log — calendar view

Full weight log page with monthly calendar (git-contribution style). Days with entries show a dot. Tap a day to filter. Custom filter chips, search, and pagination.

  • Monthly calendar with navigation arrows
  • Blue dot on days with logged entries
  • Tap day to filter list to that date
  • Filter chips: All / This Week / This Month / With Notes
  • Pagination: loads 50 entries, "Show more" for rest
  • 3-day trend delta on each row

Weight entry detail — edit & delete

Tap any weight log entry to see full details. Native iOS Form with Edit/Save toggle, weight wheel picker, date/time display, notes, and delete with confirmation.

  • Native Form sections (Weight, Date & Time, Notes)
  • Edit mode with weight wheel picker
  • Delete with confirmation dialog
  • Correct back navigation (no multi-push bug)
